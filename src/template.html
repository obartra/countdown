Countdown parameters

Text

Optional strings displayed around the countdown.

title
Page title and header text
description
Smaller text placed below the countdown
footer
Page footer text
complete
Message shown when the countdown finishes

Color

Control the background/text colors. If only one is provided, the other is inferred for contrast.

bgcolor
Background color of the page (hex or CSS color keyword)
color
Contrast color / color of the page text

Time format

Provide the countdown target time in ISO UTC:

YYYY-MM-DDTHH:MM:SSZ

Use the time query param (or date as an alias). Values in the past immediately show the complete state.

Image

Set the image param to a provider:id, chosen from the editor search. Supported providers: openverse (SVG) and tenor (stickers). Example: openverse:123e4567-e89b-12d3-a456-426614174000.

Images resolve from the provider (not bundled locally). Some providers require attribution, which is shown beneath the image.
